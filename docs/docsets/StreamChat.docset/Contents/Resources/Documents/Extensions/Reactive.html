struct Reactive<Base>
  • anyGesture(_:)

    Reactive wrapper for multiple view gesture recognizers. It automatically attaches the gesture recognizers to the receiver view. The value the Observable emits is the gesture recognizer itself.

    rx.anyGesture can’t error and is subscribed/observed on main scheduler.

    Declaration

    Swift

    public func anyGesture(_ factories: [GestureFactory]) -> ControlEvent<GestureRecognizer>

    Parameters

    factories

    a (Factory + state) collection you want to use to create the GestureRecognizers to add and observe

    Return Value

    a ControlEvent<G> that re-emit the gesture recognizer itself
